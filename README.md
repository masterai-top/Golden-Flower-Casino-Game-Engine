## 🧠 Project Overview

This is a Zhajinhua (Golden Flower) poker-style game system.  
本项目是一套炸金花（炸金花/三张牌）多人卡牌游戏系统。

Supports:

- 3-card betting game engine / 三张牌游戏引擎  
- Multiplayer table system / 多人牌桌系统  
- Hand comparison system / 牌型比较系统  
- Betting & pot system / 下注与奖池系统  
- Rule engine / 规则引擎
